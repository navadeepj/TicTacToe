# Tic Tac Toe

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. This project demonstrates core JavaScript concepts such as DOM manipulation, event handling, game logic, and state management.

## Features

- Two-player gameplay (Player O vs Player X)
- Automatic winner detection
- Draw detection
- Restart game functionality
- Clean and responsive UI
- Active player highlight
- Winning cells are highlighted

## Game Rules

- The game is played by two players.
- Player **O** always starts first.
- Players take turns placing their symbol.
- The first player to align three symbols horizontally, vertically, or diagonally wins.
- If all cells are filled without a winner, the game ends in a draw.
